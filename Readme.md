# TIL

## 목적

지식을 쌓고 기록을 남김으로서 내 PR을 좀 더 용이하게 할 수 있게 한다.

## 규칙

- 매일 그날 새로 배웠던 것에 대해 `Notion`을 사용하여 기록한다.
- 기록한 것을  `.md`파일로 내보내고 이를 `git`에 커밋한다.
- `git`에서는 주제별로 정리한다.
    - `주제\제목`으로 분류
- `git`에 잔디를 심는 목적이 아니다.
    - 매일 무리해서 작성할 필요 없다.
- 정확한 정보 전달이 목적이 아니다.
    - 일단 이해한대로 적고 추후에 알고 있던 것과 다르면 수정한다.
